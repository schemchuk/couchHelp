# AGENTS.md — Інструкції для AI-агентів

> **ОБОВ'ЯЗКОВО для KIMI CODE та всіх AI-агентів: перед початком будь-якої роботи прочитай усі `.md` файли в корені проєкту та в директоріях, що стосуються задачі.**

---

## Правила розробки

1. **Масштабованість** — кожне рішення має витримати зростання навантаження
2. **Безпека** — валідація всіх вхідних даних (zod або аналог)
3. **Git** — після кожного логічно завершеного блоку: коміт → пуш
4. **Нові модулі** — перед реалізацією створи `INSTRUCTIONS.md` у папці модуля (див. `CLAUDE.md`)
5. **Секрети** — тільки через змінні середовища, ніколи в коді
6. **Протухлі ключі** — при підозрі на недійсний ключ, попередь користувача перш ніж вносити зміни
